# 📉 Telecom Customer Churn Prediction & Interactive Dashboard

This project analyzes telecom customer data to understand churn behavior and builds an interactive dashboard to visualize key insights and support customer retention strategies.

## 🚀 Features
- Exploratory Data Analysis (EDA) on telecom data  
- Identification of churn patterns and high-risk customers  
- Feature analysis (tenure, services, billing, contract type)  
- Interactive Power BI dashboard  
- Data-driven insights for customer retention  
- Business recommendations based on analysis  

## 🛠️ Technologies Used
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  
- Power BI  

## ⚙️ Getting Started

### 🔹 Prerequisites

- Jupyter Notebook  
- Power BI Desktop
